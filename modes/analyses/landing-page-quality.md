# Landing Page Quality Analysis Mode

## 使用情境

使用者詢問 landing page 表現、paid traffic 進站品質、CVR 差、engagement 差、source / medium / campaign 對應頁面品質時使用此 mode。

## 診斷框架

Landing page quality 應以 GA4 或 onsite data 做 sanity check：

- sessions and users
- engaged sessions and engagement rate
- key events / conversions
- landing page CVR
- revenue per session
- source / medium / campaign
- device and geography if available

## 必查資料

- landing_page
- source, medium, campaign
- sessions, users, engaged_sessions
- key_events / conversions
- revenue
- device if available
- linked ad platform campaign/ad set if available

## 輸出要求

- Landing page quality summary
- Worst pages by conversion quality with sufficient traffic
- Best pages to learn from
- Paid media implication
- Tracking/data gaps
- Recommended checks

不要只因 sessions 少就判斷頁面差；需設定 minimum sample caveat。
