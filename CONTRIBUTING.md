# Contributing

Contributions of any kind are welcome!

## Reporting Issues
Please use GitHub Issues and include a clear description of the problem and steps to reproduce it.

## Submitting a PR
1. Fork this repository
2. Create a new branch (`git checkout -b feature/your-feature-name`)
3. Commit your changes
4. Open a Pull Request with a description of what you've done

## Development Setup

1. Make sure you have Python 3 installed.
2. Clone this repository
```bash
   git clone https://github.com/PoChaoWang/Digital-Marketing-Analyst-Agency.git
   cd Digital-Marketing-Analyst-Agency
```
3. Install dependencies manually based on what's imported in the scripts.
