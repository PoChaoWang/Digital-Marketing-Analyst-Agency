# Google Sheet Weekly Report Template

Use this template for reports generated by `skills/google-sheets-reporting/`.

## Spreadsheet Metadata

- Title: `Weekly Ads Performance Report - {date_range}`
- Timezone:
- Currency:
- Platforms:
- Account / property scope:
- Generated at:

## Tabs

### Config

Purpose: report scope and data assumptions.

Columns:

- field
- value
- notes

Recommended rows:

- date_range
- timezone
- currency
- platforms
- account_scope
- environment
- attribution_window
- conversion_definition
- GA4 property
- data_sources
- data_freshness
- source_types
- production_allowed

### Analysis Trace

Purpose: make the agent's analysis logic reviewable without exposing private chain-of-thought.

Columns:

- trace_type
- item
- reasoning_basis
- evidence_or_formula
- user_review_note

Recommended rows:

- source_selection
- rules_applied
- field_mapping
- metric_formula
- assumption
- decision_log

### Executive Summary

Purpose: concise leadership summary.

Columns:

- section
- summary
- risk
- owner / next step

### Platform Summary

Purpose: one row per platform or channel group.

Columns:

- platform
- spend
- impressions
- clicks
- CTR
- CPC
- CPM
- conversions
- CVR
- CPA
- revenue
- ROAS
- currency
- notes

### Campaign Detail

Purpose: campaign or ad set level performance table.

Columns:

- date
- platform
- account_id
- campaign_id
- campaign_name
- ad_group_id / ad_set_id
- ad_group_name / ad_set_name
- ad_id
- ad_name
- spend
- impressions
- clicks
- CTR
- CPC
- CPM
- conversions
- CVR
- CPA
- revenue
- ROAS
- currency
- notes

### GA4 Onsite Quality

Purpose: onsite behavior and landing page quality by source / medium / campaign / landing page.

Columns:

- source
- medium
- campaign
- landing_page
- sessions
- users
- engaged_sessions
- engagement_rate
- key_events
- event_count
- landing_page_CVR
- revenue
- notes

### Findings

Purpose: data-backed findings with separated observation, inference, and recommendation.

Columns:

- severity
- category
- observation
- inference
- reasoning_basis
- recommendation
- confidence
- risk
- supporting_rows

### Recommended Actions

Purpose: prioritized action list.

Columns:

- priority
- platform
- action
- expected_impact
- risk_level
- requires_approval
- status
- rollback_note

### Data Gaps

Purpose: missing data and limitations.

Columns:

- gap
- impact
- how_to_resolve

### Change Approval

Purpose: human-in-the-loop approval staging for proposed ad account changes.

Columns:

- current_state
- proposed_change
- reason
- expected_impact
- risk_level
- rollback_plan
- exact_changes_to_apply
- approval_required
- approval_status
